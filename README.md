# Form Validator Project

A simple client-side form validation project built using:

- HTML
- CSS
- JavaScript

## 🚀 Features

- Username validation
- Email validation (regex based)
- Password minimum length validation
- Confirm password match validation
- Error & success UI indicators

## 📂 Project Structure

form-validator/
│── index.html
│── style.css
│── script.js
│── README.md

## 🛠 How to Use

1. Clone the repository
2. Open index.html in your browser
3. Test the validation form
